## ✨ ৱেবচাইটত থকা প্ৰধান সুবিধাসমূহ (Key Features)

1. **Material Design 3 visual layout**: যিকোনো ম'বাইল, টেবলেট বা কম্পিউটাৰ স্ক্ৰীণত ধুনীয়াকৈ খাপ খাই পৰিব পৰা সম্পূৰ্ণ responsive ডিজাইন।
2. **ডাৰ্ক আৰু লাইট থিম (Dark/Light Mode)**: চকুৰ বাবে আৰামদায়ক আৰু ম'বাইলৰ চাৰ্জ বচাবলৈ ৰাতিৰ ব্যৱহাৰোপযোগী ডাৰ্ক ম'ড।
3. **পঠন সুবিধা (Font Size Scaling)**: ছাত্ৰ-ছাত্ৰী বা অভিভাৱকে নিজৰ প্ৰয়োজনমতে আখৰৰ আকাৰ সৰু-ডাঙৰ কৰাৰ বাবে `A- / A+` বুটাম।
4. **বিভাগীয় ভাগীকৰণ (Categories/Search Filter)**: গদ্য (Prose) আৰু পদ্য (Poetry) ভাগসমূহ অতি সহজে এচি বুটাম টিপিয়ে বিচাৰি ল'ব পাৰি।
5. **দ্ৰুত সন্ধান (Instant Search)**: পাঠৰ নাম সৰলভাৱে টাইপ কৰি বিচাৰিব পাৰি।
6. **আইনসংগত প্ৰশ্ন বুকমাৰ্ক (Favorites System)**: গুৰুত্বপূৰ্ণ প্ৰশ্নবোৰ বুকমাৰ্ক আইকনটো টিপি চিধাই `localStorage` ত সংৰক্ষণ কৰি ৰাখিব পাৰি।

## 🚀 ইয়াৰ জৰিয়তে ফ্ৰী ৱেবচাইট হোষ্টিং কেনেকৈ কৰিব (How to Host for Free)

আপোনাৰ টকা খৰচ নকৰাকৈ এই ৱেব ফাইলটোক লাইভ লিংক বনাবলৈ তলৰ যিকোনো এটা পদ্ধতি ব্যৱহাৰ কৰক:

### পদ্ধতি ১: Netlify ৰ সহায়ত (সটাইতকৈ সহজ)
1. [Netlify Drop](https://app.netlify.com/drop) ৱেবচাইটলৈ যাওক।
2. আপোনাৰ কম্পিউটাৰৰ পৰা আমাৰ এই **`/web`** ফ’ল্ডাৰটো (য'ত `index.html` আছে) চিধাকৈ ড্ৰেগ আৰু ড্ৰপ কৰক।
3. মাত্র ২ চেকেণ্ডতে আপোনাৰ ৱেবচাইটটো এটা লাইভ `https://...netlify.app` লিংকত প্ৰস্তুত হৈ যাব!

### পদ্ধতি ২: Vercel ৰ সহায়ত
1. [Vercel](https://vercel.com/) ত ফ্ৰী একাউণ্ট এটা বনাব।
2. তাত আপোনাৰ index.html টো আপলোড কৰক অথবা GitHub ৰ লগত সংযোগ কৰি deployment কৰি দিয়ক।

### পদ্ধতি ৩: GitHub Pages ৰ সহায়ত (বিনামূলীয়া আজীৱন হোষ্টিং)
1. GitHub ত এটা নতুন Repository খোলক।
2. তাত আমাৰ `index.html` ফাইলটো আপলোড কৰি `main` ব্ৰাঞ্চত ছেভ কৰক।
3. Repository Settings -> **Pages** লৈ গৈ `deploy from branch` ত `main` চিলেক্ট কৰক।
4. ৫ মিনিটৰ ভিতৰত আপোনাৰ ৱেবচাইটটো এতিয়া `https://username.github.io/repository-name` ত লাইভ হৈ যাব!
